def cmac_module.SUBK ( bytes  k)

Subkey generation, as in the referenced standard.

Derives two keys K1 and K2 from the input key k.

Parameters
[in]kThe key to be used. Needs to be 16 bytes long.
[out]K1Subkey #1, used to derive K2 in SUBK() and in CMAC() for messages if the message length is a multiple of AES's blocksize (128 bits).
[out]K2Subkey #2, used in #CMAC for messages if the message length is not a multiple of AES's blocksize (128 bits).
